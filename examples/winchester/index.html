Controls

Each hierarchy has its own slider so you can morph levels independently. Use the "Link sliders" checkbox to drive every layer simultaneously. Click the "Test 50%" button for a quick peek halfway along the morph. Toggles let you hide the morph layers; area circles are off by default and, when enabled, update along with the morph so you can see how the area changes.

When linked, moving one slider updates all layers
